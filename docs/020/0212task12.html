12. How to install and maintain FROST-Server on the Princeton OIT Server¶

If you are installing this on the chaos server, you can refer to these instructions from https://chenkianwee.github.io/masa3db/docs/020/020installation.html for installation. As the Princeton OIT server do not have open internet access (wget and curl do not work on the server), we have to do a few things manually for installing on the Princeton OIT server.
